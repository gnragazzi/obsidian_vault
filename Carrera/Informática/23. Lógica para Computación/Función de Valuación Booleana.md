Sean $P$ y $Q$ [[Fórmula Lógica|fórmulas]] arbitrarias, decimos que una ***valuación booleana***, o simplemente ***valuación***, es cualquier función $$v : Form \mapsto B$$ que satisfaga las siguientes condiciones:   ^bd9cb0
- V1. $v(¬P ) = ¬v(P )$  
- V2. $v(P ∨ Q) = v(P ) ∨ v(Q)$
- V3. $v(P ∧ Q) = v(P ) ∧ v(Q)$
- V4. $v(P → Q) = v(P ) → v(Q)$
