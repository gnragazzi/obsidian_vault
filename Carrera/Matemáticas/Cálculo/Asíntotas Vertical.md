La recta $x=a$ se llama la ***asíntota vertical de la curva*** $y=f(x)$ si al menos una de las siguientes es verdadera:
- $\lim_{x→a}f(x)=∞ \text{ o } \lim_{x→a}f(x)=-∞$
- $\lim_{x→a⁻}f(x)=∞ \text{ o } \lim_{x→a⁻}f(x)=-∞$
- $\lim_{x→a⁺}f(x)=∞ \text{ o } \lim_{x→a⁺}f(x)=-∞$
