Un reticulado $(A,⩽,∧,∨)$ se dice ***distributivo*** si, para cualquier terna $a, b, c∈A$ se cumple que:  
1. $a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)$,  
2. $a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)$.  
Es decir, si se cumplen las propiedades distributivas.
Si no es así, $(A,⩽,∧,∨)$ se dice que es **no distributivo**.
