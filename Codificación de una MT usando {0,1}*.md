#### Definición
- Se debe codificar una MT de tal forma que pueda ser expresada como una cadena $<M>\in\{0,1\}^*$ 
- Si $<M>$ no es una representación bien formada (según la sintaxis elegida) de una MT, se considera la representación de una [[Máquina de Turing|MT]] sin movimientos
	- De esta manera, podemos pensar en cualquier cadena binaria como una representación de una MT.
