#### Idea Informal
- Altura máxima (o camino más largo) de los árboles de computaciones para las cadenas de longitud n (aceptadas o rechazadas)

#### Definición Formal
Sea $M$ una [[Máquina de Turing no Determinística - MTND]]. La complejidad temporal de $M$ está representada por la función:  $$tc_M : \mathbb{N} → \mathbb{N}$$donde $tc_M$ es el número máximo de transiciones realizadas por una  computación (altura del árbol de computaciones) utilizando alguna secuencia de transiciones para una cadena de longitud $n$.
